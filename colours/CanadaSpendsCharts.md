# Canada Spends Chart Colours

Canada Spends displays a lot of complex data, and so needs a lot of additional colours for use in charts such as Sankeys, standard bar/pie/donut/scatter/line, and maps.

In a charting colour system, there is no real conception of "primary" over "secondary" - chart creators should feel empowered to simply select from a standard palette as they like when drafting chart views.

To that end, this document outlines various colour systems similar to the main Build Canada brand that are inspired by Canadian geography, technological innovation, and history.
