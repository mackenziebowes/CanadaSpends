Canada Spends Colours

This repository contains the colour systems for the Canada Spends project, part of the broader Build Canada brand. It provides Tailwind CSS v4 compatible colour palettes for the main brand and for complex data visualizations.

  • MainBrand.md: Details the primary 3-colour system for the main website.
  • CanadaSpendsCharts.md: Provides a wide range of palettes for charts and maps.
  • AddingNewColours.md: Offers guidance on how to create new, brand-aligned colour palettes.

Primary Colours

The main brand identity is built on a simple and serious 3-colour system composed of a light tone (Linen), a dark tone (Charcoal), and an accent (Canadia Red).

See MainBrand.md for the full palette.
